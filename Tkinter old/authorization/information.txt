#Конструкция if __name__ == "__main__": в Python используется
#для определения, запускается ли файл как основной скрипт или
#импортируется как модуль. Вот ключевые моменты:
# Определяет код, который будет выполнен только при запуске файла
# как скрипта.
# Позволяет разделить логику, которая должна выполняться при
# запуске скрипта, от той, которая должна работать при импорте
# модуля.
# Улучшает переносимость кода, позволяя использовать функции
# и классы из данного файла в других скриптах.
